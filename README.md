# groovy_fundamentals
Fundamentals for learning Groovy syntax

## What's inside
- `src/Variables.groovy`: beginner examples of Groovy variables and types
- `src/ConditionalLoops.groovy`: beginner examples of conditionals and loops

## Run the scripts
If you have Groovy installed:

```bash
groovy src/Variables.groovy
groovy src/ConditionalLoops.groovy
```
